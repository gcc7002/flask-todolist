# Flask To-Do List

A basic to-do list web app built with Flask and SQLite. You can add tasks with a title, optional description and tags. Tasks can also be marked as completed or deleted.

---

## Features

- Add tasks
- View all tasks
- Delete tasks
- Mark tasks as completed 
- Edit tasks (WIP)
